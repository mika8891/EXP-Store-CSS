# EXP Game Store

A Laravel-based game storefront with HTML/CSS frontend and Blade views for authentication and purchase flow.

## Features
- Login & Registration (Blade views)
- Game Storefront
- Shopping Cart & Purchase

## Setup

1. Install dependencies:
   \\\ash
   composer install
   \\\

2. Copy .env:
   \\\ash
   cp .env.example .env
   \\\
3. Generate app key:
   \\\ash
   php artisan key:generate
   \\\

4. Run dev server:
   \\\ash
   php artisan serve
   \\\

Open http://127.0.0.1:8000
